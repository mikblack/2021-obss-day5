This workshop is designed to be run on the NeSI compute infrastructure.

## Software

In order to run the workshop on your own computer you will need to ensure you installed:

- A UNIX shell such as BASH
- R and RStudio
- R packages
  - `dplyr`
  - `limma`
  - `edgeR`
  - `goseq`
  - `beeswarm`
